# 🕰️ Old iPad Desk Clock (구형 아이패드 탁상시계)

![Built with Gemini](https://img.shields.io/badge/Co--coded%20with-Gemini-4E86F8?style=flat-square)

iOS 11 구형 아이패드(iPad Air, Mini 2 등)를 감성적인 디지털 탁상시계로 활용하기 위한 웹 애플리케이션 프로젝트입니다.
오래된 기기의 사파리 브라우저에서도 부드럽게 작동하도록 최적화되었습니다.

## ✨ 주요 기능 (Features)

* **📱 구형 기기 완벽 지원:** iOS 11 Safari 및 구형 WebKit 엔진 호환성 확보
* **🎨 감성적인 디자인:**
    * 오전(오렌지) / 오후(스카이블루) 테마 자동 변경
    * 배경의 은은한 '심호흡(Breathing)' 애니메이션
    * 미니멀한 나눔고딕 폰트 디자인
    * 부드러운 페이드 인(Fade-in) 숫자 전환 효과
* **⚙️ 사용자 설정 (UI Control):**
    * 화면 터치 없이 우측 하단 버튼으로 설정 메뉴 호출
    * 시계 크기(Size), 가로/세로 위치(Position) 조절 가능
* **⚡ PWA 지원:** 홈 화면에 추가 시 주소창 없이 전체 화면 앱처럼 실행
* **🚫 화면 꺼짐 방지:** NoSleep.js를 활용한 화면 상시 켜짐(Always-on) 기능 내장

## 🛠️ 기술 스택 (Tech Stack)

* **HTML5 / CSS3 / JavaScript (Vanilla JS)**
* **NoSleep.js:** 화면 자동 잠금 방지
* **GitHub Pages:** 정적 웹 호스팅

## 🚀 사용 방법 (How to use)

1.  아이패드 사파리(Safari) 브라우저를 엽니다.
2.  이 저장소의 GitHub Pages 주소로 접속합니다.
    * `https://[본인아이디].github.io/[저장소이름]/`
3.  상단(또는 하단) **[공유]** 버튼을 누릅니다.
4.  **[홈 화면에 추가]**를 선택합니다.
5.  홈 화면에 생성된 아이콘을 눌러 실행합니다.
6.  우측 하단 **⚙️ 설정 버튼**을 눌러 '화면 켜짐 유지'를 켜주세요.
